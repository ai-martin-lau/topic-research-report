# 深度调研报告模板

复制本模板结构到 `调研报告-中文.md` 和 `research-report-en.md`，填充时删除占位说明。中文文件只保留中文内容，英文文件只保留 English 内容。两版信息一致，但不要逐句硬翻译。

报告是完整研究报告，不是表单、改编稿、制作稿或素材清单。标题后直接进入正文，不要用「选题定义」「研究结论」「执行摘要」这类元信息开头。
